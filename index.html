    Alma Acquisitions - Open Source

    AAOS displays your library's latest acquisitions: e-books, journals, e-journals, etc. and allows the library's employees to manage the acquisitions lists via an clear and user-friendly admin interface.

    Get started Go to GitHub

    All your library's acquisitions, from Alma!

    Display the latest acquisitions to your patrons by importing acquisitions data directly from Alma

    Open Source

    AAOS is just a starting point. Something does not fit your needs? Feel free to modify and change anything. Added an awesome feature? Consider submitting your code to this repository!

    Easy to manage

    New acquisitions lists? Old lists that must be disabled? All can be managed directly in AAOS admin panel!
